Pre-requisettes

It is assumed that you already have the following installed:

  • python 2.5 http://www.python.org/
  • setuptools http://peak.telecommunity.com/DevCenter/setuptools

Reference

In the documentation below, the following symbols will be used in leau of concrete examples

  • $PROJECT: the name of the project that we will create ex: myfirstapplciation

Get the distribution

Download the java sdk distribution from http://www.appcelerator.org

  • Appcelerator-@VERSION@-py2.5.egg - this is the distribution for creating your python services

Easy install

Lets install Appcelerator 

easy_install Appcelerator-@VERSION@-py2.5.egg
Now lets install Pylons (will be used later for creating the project) 
easy_install Pylons
And you will need sqlalchemy (beaker requires it) 
easy_install sqlalchemy

Create your project

paster create -t appcelerator $PROJECT
this will generate your project for you in a subdirecty named $PROJECT

start the server

paster serve --reload development.ini
You should now be able to open http//localhost:5000

Testing

You should now be able to hit the test service. Goto http://localhost:5000/servicetester.html and put the following in
  • message type: remote:app.test.message.request.response
you should get the following back
  • message type: remote:app.test.message.response
  • data.message: "I received from you: Hello,world"
  • data.success: "true"

Python Service

To add a new service, simply properly define methods in a class as the individual handlers. The key things for this are that
  • the file ends with _service.py
  • you put the proper method signature method_name(message)
  • you put the proper annotations directly above the method @Service(request=xxx,response=xxx)

Web Implementation

Once you have created the java project, your web source will exist in:
  • src/web: directory where all your html exists
  • src/web/appcelerator.xml: defines the location and behaviour of the service broker
  • examples/simple/src/web/index.html: the sample source used for this demonstration
