Validation

The Validation widget allows you to build a state machine around validation. As members of the Validation widget are being validated you can publish messages. This allows you to publish a message when all fields are valid or when one or more fields are invalid.

Required Options:

on The expression that allows you to publish a message when all fields are valid or when at least 1 field is invalid. When all fields are valid you can use the built-in condition valid

Usage: on="valid then l:fields.valid[valid=true] else l:fields.valid[valid=false]"

members elements A space-delimited list of the element IDs that should be part of the Validation widget. Each element ID must have a validator defined

Usage: <members> field1 field2 field3 </members>
