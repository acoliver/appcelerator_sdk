The folder widget has both <folder> and <item> elements for which they both have the same required and optional attributes.

Required Options:

id the identifier for the widget
<folder> name the displayed name of a folder
<item> onopen hanlder for the open of the item : where you usually put your message
<item> TEXT the innerHTML for the <item> element is the displayed name for the item

Optional Options:

<folder> or <item> opened_icon location of the image to use for when item or folder is open
<folder> or <item> closeed_icon location of the image to use for when item or folder is closed
