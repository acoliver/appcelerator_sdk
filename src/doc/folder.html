The folder widget has both <folder> and <item> elements for which they both have the same required and optional attributes.

Required Options:

name The display name of a folder
<item> TEXT </item> The innerHTML for the <item> element is the display name for the item

Optional Options:

onopen The message to be publish when the item is selected
open A condition which will trigger the item to be opened
opened_icon Location of the image to use for when item or folder is open. This attribute can be applied to the Folder and the Item
closed_icon Location of the image to use for when item or folder is closed. This attribute can be applied to the Folder and the Item
