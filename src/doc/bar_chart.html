Required Options:

on The message that contains the data to populate the chart
type The chart type: bar, line, pie or dynamic property value
property The array property that contains the chart data

Optional Options:

color Specifies the base color of the chart or the color for each bar/slice/line - defaults to #477398. You can provide a single hex color or csv color list
title Title displayed at the top of the chart. Default is no title.
angle Angle for '3D' graphs (bar and pie). Default is 15 (in degrees)
thickness Thickness of '3D' graphs. Default is 15 (in pixels)
width The width of the chart. The default is 400 (in pixels)
height The height of the chart. The default is 360 (in pixels)
chartMode The sub-type for bar and line charts. Defaults to "clustered" for bar charts. Defaults to "line" for line charts. Other values: "stacked" and "100% stacked"
barOrientation "vertical" or "horizontal" orientation for a bar chart
rotateXAxisLabel "true" or "false". If "true" x axis labels are rotated by 90 degrees. Default is false.
rotateYAxisLabel "true" or "false". If "true" y axis labels are rotated by 90 degrees. Default is false.
legend "true" or "false". If "true" a legend will be displayed. Default is false.
textSize Font size. Default is 11 (in pixels)
chartTitles Array property name that contains the titles for the legend and labels composite bar charts.
fillAlpha Specifies the opacity of the shaded area under a line in a line chart. Default is 30 (in %)
indicator Enable a mouseover bubble to show the value of every line at a given x value on a line graph. Default is 'false'
oneBalloon Specifies whether an indicator balloon will be shown for all lines or only the line closest to the mouse pointer on the line graph. Possible values are 'true' or 'false'. Default is 'true'
marginTop Specifies the space between the top of the chart and the start of the line or bar chart. Default is 50 (in pixels)
marginLeft Specifies the space between the left of the chart and the start of the line or bar chart. Default is 50 (in pixels)
marginRight Specifies the space between the right of the chart and the start of the line or bar chart. Default is 50 (in pixels)
marginBottom Specifies the space between the bottom of the chart and the start of the line or bar chart. Default is 50 (in pixels)
legendHighlight Specifies whether the lines on the line chart will be "highlighted" when one mouses over a legend value for that line. Default is "true"
innerRadius Specifies the radius of the "hole" in the center of the pie chart.
