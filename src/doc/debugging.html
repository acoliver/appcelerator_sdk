Debugging and Logging

The Appcelerator Platform allows you view debug level information from the platform and from your application, including messages that are sent and received by your application.

Currently, the Appcelerator Platform only logs messages to the Firebug console in Firefox, so you must run Firefox and install the Firebug plugin.

Debug Mode

To enable Debug mode, simply add ?debug=1 to the end of your URL and open the Firebug console to see all debug statements.
