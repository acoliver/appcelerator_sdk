Debugging

Throughout the framework, it is possible to view debugging statements by adding the debug=1 query parameter. Viewing the console is subject to which browser you are using. In firefox for example, you can open the firebug window and click the Console tab. For example:

file://localhost/Users/azuercher/workspace/appcelerator_sdk/stage/doc/index.html?debug=1

Logging

To publish your own logging statements you can use the following syntax The following levels are currently supported

  • info
  • warn
  • error
  • debug
  • trace
  • fatal

Logger.info('hi info');
