Content

Example | Code | Usage

The content widget is useful for segregating your application into individual pages. This is useful as it allows for:

  • decomposition of your application into individual sets for developers to work on
  • the lazy loading of portions of the application
  • tying application content to a state machine to conditionally display segmented content
  • pass parameters into content for reuse
