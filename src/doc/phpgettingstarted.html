PHP

Pre-requisettes

It is assumed that you already have the following installed:

  • apache
  • php5 support for apache (ex: http://www.entropy.ch/software/macosx/php/)

Get the distribution

Download the java sdk distribution from http://www.appcelerator.org

  • appcelerator_sdk_php_2.0.0.zip - this is the distribution for creating your php services

Extract

unzip appcelerator_sdk_php_2.0.0.zip -d appcelerator_sdk_php_2.0.0
cd appcelerator_sdk_php_2.0.0

Deploy

If you are running apache with the php5 support then you can simply copy to your sites directory 

cp -R src/web/* ~/Sites
cp -R src/php/* ~/Sites
Now you can hit the test page (replace myname with your actual username): http://localhost/~myname/index.html

Testing

You should now be able to hit the test service. Goto http://localhost/~myname/servicetester.html and put the following in
  • message type:remote:app.test.message.request
you should get the following back
  • message type:remote:app.test.message
  • data.message:"I received from you: Hello,world"
  • data.success:"true"
If you look in the following you can see how to implement services
  • src/php/services/sample_service.php
You can add new services by creating files in the services directory and making sure that they end with _service.php and follow the same method declaration (including the comments).
