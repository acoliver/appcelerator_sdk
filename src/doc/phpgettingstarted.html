Installation | Web Client | PHP Service

Pre-requisites

It is assumed that you already have the following installed:

  • Apache 1.3 or higher
  • PHP 4/5

Reference

In the documentation below, the following symbols will be used in lieu of concrete examples

  • $APACHE_USERHOME: ex: /Users/azuercher/Sites
  • $USER: ex: azuercher

Get the distribution

Download the PHP SDK distribution from http://www.appcelerator.org

Deploy

If you are running apache with the php support then you can simply copy to your sites directory 

cp -R src/web/* $APACHE_USERHOME
cp -R src/php/* $APACHE_USERHOME
Now you can hit the test page (replace myname with your actual username): http://localhost/~$USER/index.html

Testing

You should now be able to hit the test service. Goto http://localhost/~$USER/servicetester.html and put the following in
  • message type: remote:app.test.message.request.response
you should get the following back
  • message type: remote:app.test.message.response
  • data.message: "I received from you: Hello,world"
  • data.success: "true"

Next Steps

You are now ready to proceed to creating your Web Client

PHP Service

To add a new service, simply properly define methods in a class as the individual handlers. The key things for this are that
  • the file ends with _service.php
  • the class ends with Service
  • you put the proper method signature method_name(&$request,&$response)
  • you put the proper comments directly above the method @Service(request=xxx,response=xxx,version=xx)

Troubleshooting

On apache, you can view the logs to see what exceptions may occur based upon your ErrorLog config (/etc/httpd/httpd.conf) 
ErrorLog "/private/var/log/httpd/error_log"
You may want to restart apache (rarely needed)

Web Implementation

Once you have created the java project, your web source will exist in:
  • src/web: directory where all your html exists
  • src/web/appcelerator.xml: defines the location and behaviour of the service broker
  • examples/simple/src/web/index.html: the sample source used for this demonstration
