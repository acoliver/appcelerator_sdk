Datatable

A Datatable provides a simple way for build a standard read-only HTML table. The Datatable also supports string-based sorting by simply clicking on the table column headers.

Required Attributes:

on The handler that controls the activation of the widget usually of the form ">dataevent< then execute"
property The array property that contains the table data

Optional Attributes:

sort The sort mode used by the data table. Possible values are 'client', 'server', and 'none'; if server is specified, then sortRequest must also be specified. Defaults to client sort
sortRequest The sort message sent by the datatable to the corresponding service when a header is clicked. The message contains two values in its payload: 'property' and 'direction'. The property value is the name of the header property by which the table should be sorted. The direction is either 'asc' or 'desc' for ascending or descending orders, respectively.
width The percentage the width of the table relative to its container. Default is 100%
rowEvenClass The CSS class to be used on even rows of the table
rowOddClass The CSS class to be used on odd rows of the table

Required Header Definition:

header elements The set of elements specifying the header property, text, alignment and width
property The array property that contains the column data
text The text to be placed in the header cell. This is the value of the header element.

Optional Header Definition:

width Specifies the width of a given column
align Specifies the alignment of the text in each cell of a given column

Optional Style Definition (simply override the following CSS classes):

table_row The style applied to each html tr element table row
table_cell The style applied to each html td element table cell. This is the default class for cells if rowOddClass or rowEvenClass are not provided
table_row_header The style applied to each html tr element header row
table_cell_header The style applied to each html td element header cell
