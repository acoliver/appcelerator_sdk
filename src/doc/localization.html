Localization

Example | Code

To add localication to your application, simply

  • define an attribute named "langid" on the element for which you would like the inner html to have the localized resource. This will cover a majority of the scenarios where the localization is a simple one for one replacement
  • define your resource bundles in a javascript file and call Appcelerator.Localization.addLanguageBundle
  • for more complex formatting call Appcelerator.Localization.getWithFormat to render the message

formatted localization var message = Appcelerator.Localization.getWithFormat('localization.example.message',null,null,this.data); $('localcation_example_span').innerHTML = message;
