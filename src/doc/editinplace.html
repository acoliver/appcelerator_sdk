Required Options:

saveOn "on" expression for when the user clicks Save
cancelOn "on" expression for when the user clicks Cancel

Optional Options:

defaultValue Initial value for the field.
validator The name of the validator for the field
message Appcelerator message name that will contain the data to populate the field dyamically
property Appcelerator message property that contains the value for the field
type The form control type (defaults to text)
position position for the div element (defaults to "right"). Valid values are: top, left, right, bottom
errorPosition position for the div element if an error occurs (defaults to "top"). Valid values are: top or bottom
errorMessage Custom error message displayed when the specified Validator fails
defaultClassName the css class used to style the text in non-edit mode
editClassName the css class used to style the input field while in edit mode
buttonClassName the css class used to style the buttons while in edit mode
