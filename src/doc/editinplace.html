Edit-in-Place

Example | Code | Usage

This widget allows for the direct mutation of text to toggle between editing and standard text display.

Edit the text below and click 'save'. You should see a passed message below with the value of the text.

alert("you saved"); alert("you cancelled");

Here's the code:

Required Options:

id the identifier for the dom element
saveOn the handler for when the Edit-in-Place widget is saved
cancelOn the handler for when the Edit-in-Place widget is cancelled
validator the name of the validator for the field
defaultValue initial value for the control

Optional Options:

type The form control type (defaults to text)
position position for the div element (defaults to right)
errorPosition position for the div element if an error occurs (defaults to top)
defaultClassName the css class whilst idle (defualts to '')
editClassName the css class whilst editing (defaults to '')
buttonClassName the css class for the save and cancel buttons (defaults to '')
