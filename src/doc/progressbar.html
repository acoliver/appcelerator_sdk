Required Options:

message the message that will trigger an update to a progress bar widget.
property the property name under which the progress bar widget will look for the new value. Values passed to a progress bar should be numeric and between 0 and 100 inclusive.

Optional Options:

borderClassName the class name of the border portion of the progress bar. This allows the border to be styled via CSS.
fillClassName the class name of the fill portion of the progress bar. This allows the fill to be styled via CSS.
width the width progress bar widget, including units. The default width is 500px.
height the height progress bar widget, including units. The default height is 20px.
animate this attribute determines whether or not changes to the progress bar value are animated. Can be true or false (defaults to true).
