Required Options:

on Any on expression will work here. The button additionally supports the 'enable' and 'disable' actions.

Optional Elements:

width Width of the button. Defaults to 200px.
corner Set to true or false (for square). Defaults to true.
color Set to dark or light. Defaults to dark.
icon Supports add_green, add_gray, delete_red, delete_gray, edit_yellow, edit_gray, save_blue, save_gray attributes. Defaults to none.
