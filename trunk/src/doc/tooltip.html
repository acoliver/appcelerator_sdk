Tooltip

Example | Code | Usage

The Tooltip widget allows you to easily construct and display nice visual tool tips over an element when you hover over it. The widget automatically controls the positioning and visibility of the tooltip. The content of the tooltip container is specified inside the definition of the tooltip using standard HTML elements. Required Options:
element the target element to trigger the tooltip once the mouse is over the element
Optional Options:
backgroundColor background color for the tooltip container. defaults to #FC9 if not specified.
borderColor border color for the tooltip container. defaults to #c96 if not specified.
textColor foreground color for the tooltip container. defaults to #111 if not specified.
mouseFollow should the tooltip follow the mouse as it moves over the element. defaults to true.
opacity opacity for the tooltip container. defaults to .75 if not specified.
