The Appcelerator Platform allows you to build RIAs without using Javascript, but we certainly do not mind if you want to write Javascript. To make this easy for you, we created a Script Widget. Within a Script Widget, you can write any valid Javascript code. The Script Widget is powerful because it allows you to trigger the execution of Javascript using our Expression Language like so on="[condition] then execute"

Optional Options:

on If the Script Widget does not have a on expression defined. The script will be executed when the page is loaded. If the expression is defined the script will be executed when the on condition is met.

Usage: on="[condition] then execute".

You can add an optional after that will execute the script when its condition is met after some defined amount of time - e.g., on="l:my.message then execute after 2s"

inner HTML The actual javascript itself. Note that since this is in the html, you must properly escape the script for example using &lt; instead of <.
