Options:

Param Description Default
header_text The text that should be displayed in the header. Setting a value implies a header will be shown blank
color The color scheme of the widget. Supported schemes: light_gray, dark_gray dark_gray
rounded Set to true for rounded corners false
close Set to true to display a close button false
shade Set to true to enable shade/unshade. Setting this implies the header will be shown false
on Set to enable the widget to listen to events none
draggable Set to enable dragging false
resizable Set to enable the widget to be resized false
tail Set to right or left to show a tail under the box (speech bubble) none
width Width of the dialog box unset
height Height for the dialog box unset
userClass Additional classes for the dialog none

Enabling a header:

The header bar is automatically created when when either the header_text parameter is set or shade is set to true.
