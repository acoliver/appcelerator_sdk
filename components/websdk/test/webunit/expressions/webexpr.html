Web Expressions

These tests exercise web expressions parameter matching.

l:test[count>0]
l:test[count>1]
l:test[count<101]
l:test[count<=100]
l:test[count>=100]
l:test[!count>101]
l:test[!count>=101]
l:test[!count<101]
l:test[!count<=200]
l:test[count!=200]
l:test[count=100]
l:test[count=~100]
l:test[count=~1]
l:test[count=~^1]
l:test[count=~0$]
l:test[!count=~9$]
l:test[count]
l:test[!foobar]
l:test[count,!foobar]
